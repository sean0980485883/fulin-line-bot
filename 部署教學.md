# LINE 翻譯機器人部署教學

## 📁 檔案結構

```
line-translator-bot/
├── index.js          ← 主程式
├── package.json      ← 套件設定
└── .env.example      ← 環境變數範例
```

---

## 第一步：建立 LINE Bot 帳號

1. 前往 https://developers.line.biz 登入（用 LINE 帳號）
2. 點擊 **Create a new provider** → 輸入名稱（例如：翻譯機器人）
3. 點擊 **Create a new channel** → 選擇 **Messaging API**
4. 填寫：
   - Channel name：翻譯機器人 🌐
   - Channel description：自動偵測語言翻譯
   - Category / Subcategory：任選
5. 建立完成後，進入 **Messaging API** 分頁

### 取得金鑰

| 項目 | 在哪裡找 |
|------|---------|
| Channel Secret | Basic settings 分頁 |
| Channel Access Token | Messaging API 分頁 → 最下方 Issue |

### 開啟群組功能

在 **Messaging API** 分頁中：
- **Allow bot to join group chats** → 開啟 ✅
- **Auto-reply messages** → 關閉 ❌
- **Greeting messages** → 關閉 ❌

---

## 第二步：取得 Claude API Key

1. 前往 https://console.anthropic.com
2. 點擊 **API Keys** → **Create Key**
3. 複製保存（只顯示一次）

---

## 第三步：部署到 Render（免費）

### 3-1. 上傳程式碼到 GitHub

```bash
git init
git add .
git commit -m "LINE 翻譯機器人"
git remote add origin https://github.com/你的帳號/line-translator-bot.git
git push -u origin main
```

### 3-2. 在 Render 建立服務

1. 前往 https://render.com → 登入 → **New Web Service**
2. 連結你的 GitHub repo
3. 設定：
   - **Name**：line-translator-bot
   - **Runtime**：Node
   - **Build Command**：`npm install`
   - **Start Command**：`npm start`
   - **Instance Type**：Free

### 3-3. 設定環境變數

在 Render 的 **Environment** 分頁新增：

```
LINE_CHANNEL_SECRET     = 貼上你的 Channel Secret
LINE_CHANNEL_ACCESS_TOKEN = 貼上你的 Token
ANTHROPIC_API_KEY       = 貼上你的 Claude Key
```

### 3-4. 部署並取得網址

部署完成後會得到類似：
```
https://line-translator-bot-xxxx.onrender.com
```

---

## 第四步：設定 Webhook

1. 回到 LINE Developers Console → Messaging API 分頁
2. **Webhook URL** 填入：
   ```
   https://line-translator-bot-xxxx.onrender.com/webhook
   ```
3. 點擊 **Verify** → 應看到 ✅ Success
4. **Use webhook** → 開啟 ✅

---

## 第五步：加入群組測試

1. 用 LINE 掃描 QR Code（在 Messaging API 分頁）
2. 加為好友
3. 在群組中邀請機器人
4. 傳送任意語言的文字，機器人會自動翻譯 🎉

## 常見問題

**Q: Render 免費版有什麼限制？**
A: 免費版 15 分鐘無流量會休眠，第一則訊息可能延遲 30 秒。可升級 $7/月 方案保持常駐。

**Q: 機器人在群組翻譯所有人的訊息？**
A: 是的，目前設計是翻譯所有文字訊息。若只想翻譯特定格式（如 `!翻譯 xxxxxx`），可修改 index.js 的事件處理邏輯。

**Q: Claude API 費用大概多少？**
A: claude-sonnet-4 約 $0.003/1000 tokens。一般聊天群組一個月費用大約 $1–5 美金。

## 本機測試（可選）

```bash
# 安裝套件
npm install

# 複製環境變數
cp .env.example .env
# 編輯 .env 填入金鑰

# 安裝 dotenv
npm install dotenv

# 在 index.js 最頂端加入
require('dotenv').config();

# 啟動
npm run dev

# 用 ngrok 暴露本機 port 給 LINE
npx ngrok http 3000
```
